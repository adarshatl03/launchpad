# NON-GOALS

This document defines what **AtoZyx LaunchPad is explicitly NOT**.

If a proposal or pull request conflicts with these points, it will be rejected.

---

## Not a Task or Project Management Tool

LaunchPad does not manage tasks, tickets, or sprints.

## Not a Docs or Wiki Platform

LaunchPad enforces structure and avoids free-form documentation.

## Not an Idea Generator

LaunchPad does not generate ideas or validate markets.

## Not an AI Chat Tool

All logic is deterministic and explainable.

## Not a Pricing or Estimation Engine

No guarantees on timelines or costs.

---

**Final Rule:** If a feature reduces clarity, it does not belong.
